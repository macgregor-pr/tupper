# Tupper's Self Referential Formula
This repository contains a javascript implementation of Tupper's self referential formula which is described in <a href=https://www.youtube.com/watch?v=_s5RFgd59ao>this</a> excellent video on the numberphile youtube channel. This mini project is featured on my website at www.pmacg.co.uk.
